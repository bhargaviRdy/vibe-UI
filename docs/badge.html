Badges

Badges are used when we need to show an extra information like status or a number or a tag showing new. We can use Badges along with avatars for different sizes using badge-xl, badge-lg, badge-sm, badge-xsclasses. We can also show avatars in different colours using badge-red, badge-green, badge-orange classes. We can use a number badge using badge-number class. avatar-xs, avatar-sm, avatar-lg, avatar-xl, avatar-text classes to our code.
